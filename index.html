What We Do

Our application is an open source Internet of Things (IoT) application and API to store and retrieve data from things using the HTTP protocol over the Internet or via a Local Area Network. The application enables the creation of sensor logging applications, location tracking applications, and a social network of things with status updates".

This applicaion is built to fulfill the criterias of the class assignment given to us in Enterprise Information Systems module.
